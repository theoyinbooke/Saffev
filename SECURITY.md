# Security Policy

Saffev is an **on-device** tool. By design, nothing leaves the device: it makes
no network calls except to the local inference engine it supervises, stores data
locally in an encrypted-at-rest SQLite database, and never sends content
anywhere.

## Reporting a vulnerability

Please report security issues **privately**. Do not open a public issue for
vulnerabilities.

Email: **horllsey@gmail.com**

Include a description of the issue, steps to reproduce, and the affected
version/commit if known. We will acknowledge your report and work with you on a
fix and a coordinated disclosure.
